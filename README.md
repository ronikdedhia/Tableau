# Tableau

A collection of Tableau workbooks for data analysis and visualisation across diverse domains. Each `.twb` file is a self-contained dashboard exploring a different dataset — from global vehicle sales to US unemployment trends.

## Dashboards

| Workbook | Domain |
|----------|--------|
| Airline Blend | Aviation — flight routes and performance |
| Amazing Store - Map | Retail geospatial sales analysis |
| Annual Bonus Analysis | HR — bonus distribution by department |
| Bank | Banking — customer and transaction analytics |
| Joins | Data modelling — join type demonstration |
| Laundry Regions | Regional operational analysis |
| Unemployment Rate US | US macroeconomic unemployment trends |
| Vehicles Sold Globally | Global automotive sales by region and brand |

## Getting Started

### Prerequisites

- Tableau Desktop or Tableau Public ([download](https://public.tableau.com/en-us/s/download))

### Open a Dashboard

```bash
git clone https://github.com/ronikdedhia/Tableau.git
cd Tableau
```

Double-click any `.twb` file to open it in Tableau Desktop, or import it into Tableau Public.

> **Note:** If the workbook references an external data source, reconnect to the data file when prompted.
